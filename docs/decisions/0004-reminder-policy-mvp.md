# ADR-0004: 리마인더 MVP 정책 (H6 추천안 A)

## 상태
승인 (스테이징·MVP)

## 결정
PRD P-N4 전체(72h·48h·8회) 대신 **마감 48h / 24h / 1h** 구간별 **인앱 1건 + FCM 1건**으로 시작한다.

## 규칙
- **P-N10**: 해당 단계에서 이미 투표·참석 응답을 완료한 사용자는 스킵
- **P-N3**: 22:00~08:00 KST에는 알림 생성·푸시를 **08:00**으로 이월
- **중복 방지**: 동일 이벤트·사용자·구간(48h|24h|1h)당 24시간 내 1회
- `autoReminderEnabled=false` 사용자 제외

## 후속
파일럿 피드백 후 PRD 전체 스케줄로 확장 가능.
