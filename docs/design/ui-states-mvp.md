# UI 상태 매트릭스 (MVP)

**버전:** 0.2  
**근거:** PRD §7, §8.7 (X-1~X-7), product-ui-core

범례: ● 필수 구현 · ○ MVP 단순화 가능

---

## 공통 패턴

| 상태 | UI 패턴 | API/조건 |
|------|---------|----------|
| 로딩 | 스켈레톤 또는 중앙 스피너 | 요청 중 |
| 빈 | 일러스트 + 안내 + 1차 CTA | `items.length === 0` |
| 오류(네트워크) | 배너 + 재시도 | timeout, offline (§8.8) |
| 오류(서버) | 토스트 + 재시도(PUT/POST) | 5xx |
| 오류(4xx) | 메시지 매핑 (X-4) | code별 |
| 권한 | “권한이 없어요” + 뒤로 | 403 |
| 충돌 | 다이얼로그 + 새로고침 | 409 VERSION_MISMATCH |
| 읽기전용 | 입력 비활성 + 안내 칩 | cancelled, closed |
