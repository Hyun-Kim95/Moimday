# 화면 스펙 (MVP)

**버전:** 0.2  
**근거:** PRD §5~§7, F1~F6  
**플랫폼:** iOS·Android 동일 IA

---

## 네비게이션 (IA)

```
[탭] 홈(S04) | 모임(S05) | 일정(S09) | 알림(S10) | 설정(S11)
         └─ 모임 상세(S07) ─┬─ 투표 집계(S13)
                            ├─ 확정 일시(S14) [생성자]
                            └─ 응답 시트(S08)
[모달/스택] 로그인(S02) · 그룹(S03) · 모임 생성(S06) · 도움말(S12)
```

딥링크: `moimday://invite/{token}`, `moimday://events/{id}`

## 화면별 스펙

### S01 스플래시·온보딩
| 항목 | 내용 |
|------|------|
| 진입 | 앱 cold start |
| 목적 | 가치 제안 1~2장 + 푸시 권한 안내 |
| CTA | 시작하기 → S02 또는 (로그인됨) S04/S03 |
| 상태 | 권한 거부 시 S12 링크 |

### S02 전화번호·OTP
| 항목 | 내용 |
|------|------|
| 필드 | 휴대폰 번호, OTP 6자리 |
| 검증 | P-A4 형식, P-A1~A3 |
| CTA | 인증번호 받기 → 확인 → (hasGroup) S04 else S03 |
| 약관 | 만 14세 이상 체크 (P-E5) |

### S03 그룹 생성·초대
| 항목 | 내용 |
|------|------|
| 분기 | 내 그룹 목록·전환 / 새 그룹 만들기 / 초대 참여 |
| 표시 | 멤버 N/30, 대기 초대 수 |
| CTA | 카카오·링크 복사 공유 |

### S00 그룹 전환 (홈·모임 앱바)
| 항목 | 내용 |
|------|------|
| 조건 | 소속 그룹 2개 이상 |
| 동작 | 바텀시트 선택 → `PATCH /users/me/active-group` |

### S04 홈
| 항목 | 내용 |
|------|------|
| 섹션1 | **내가 할 일** — 배지 `투표 필요` / `참석 답변` |
| 섹션2 | **다가오는 일정**(finalized) |
| 섹션3 | **그룹 미완료** — 모임별 이름(최대 4줄 요약 + 전체 보기) |
| 상태 | 빈: “예정된 일정이 없어요” + 모임 만들기 |

### S05 모임 목록
| 항목 | 내용 |
|------|------|
| 필터 | 전체 / 내 미완료 |
| 행 | 제목, 상태칩, 마감 D-day |
| FAB | 모임 만들기 → S06 |

### S06 모임 생성·편집
| 항목 | 내용 |
|------|------|
| 모드 토글 | 후보 투표 / 일시 확정 |
| 투표 | 후보 2~5, 투표 마감 |
| 확정 | 일시 1개, 참석 마감 |
| 검증 | P-E7 클라이언트 선검증 |
| 저장 | POST events → S07 |

### S07 모임 상세
| 항목 | 내용 |
|------|------|
| 헤더 | 제목, 상태, 일시·장소 |
| 본문 | 단계별 CTA(투표/참석), 집계 링크(S13), 미완료 명단 |
| 생성자 | 확정 일시(S14), **투표 마감 연장**, 최종 확정, 취소, 수동 독촉(일 1회) |
| 생성자 부재 | `ORGANIZER_UNAVAILABLE` — 편집·확정 비활성(P-E15) |
| 하단 | 댓글 목록 + 입력(F6) |
| 읽기전용 | `cancelled`, 마감-only 조회 |

### S08 응답 바텀시트
| 항목 | 내용 |
|------|------|
| 투표 | 후보별 가능/불가/미정 세그먼트 |
| 참석 | 참석/불참/미정 + 메모(선택) |
| 제출 | `PUT` + `Idempotency-Key`, 실패 시 재시도 |

### S13 날짜 투표 집계
| 항목 | 내용 |
|------|------|
| 표시 | 후보별 득표, 가능/불가/미정/미응답 명단(8명 미리보기 + 전체 보기) |
| 빈 | 아직 투표 없음 |

### S14 확정 일시 선택 (생성자)
| 항목 | 내용 |
|------|------|
| 목록 | 후보 + 가능 인원 수 |
| 0명 가능 | 경고 다이얼로그 (P-E17) |
| CTA | 이 일시로 확정 |

### S09 스케줄
| 항목 | 내용 |
|------|------|
| 뷰 | 월 그리드 + 리스트(권장 둘 다) |
| 항목 | finalized만 |
| 탭 | 일정 → S07 |

### S10 알림
| 항목 | 내용 |
|------|------|
| 행 | 유형, 제목, 시각, 읽음 |
| 탭 | 관련 모임 S07 |

### S11 설정
| 항목 | 내용 |
|------|------|
| 항목 | 프로필, 자동 리마인더 on/off, **계정 삭제**, 그룹 나가기, 도움말 |
| 관리자 | 관리자 이관(`POST .../admin/transfer`), 멤버 보내기, 그룹 해체 |

### S12 도움말
| 항목 | 내용 |
|------|------|
| 내용 | 푸시·배터리 최적화(Android) 안내 |
